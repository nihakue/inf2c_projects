Getting it to run

Steps taken
  1. Fork the django-cms repo on github
  2. Clone the source to my mac
  3. Discover that the version of python that ships with Mac is shit
  4. install homebrew (package management)
  5. use homebrew to install python 2.7.5
  6. install pip (python module management)
  7. use pip to install virtualenv and virtualenvwrapper
  8. create a virtual environment in django-cms/
  9. pip install all the django-cms dependencies
  10. run the end-to-end django-cms test
  11. modify/run the project creation source script to create a custom new project (pictured)
